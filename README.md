## TIP:
 1. 当配置安装U盘的时候，写镜像的软件可能会报错（1 device failed），无视继续重启进入安装即可，U盘需要8G以上的8G不够，最好使用好点的U盘速度快成功率会高一些
  
 2. 配置完安装U盘时候，最后一次安装系统（进入用户设置那次，语言等）偶尔会出现灰色屏幕，只有右上角有鼠标指针，重启重新操作一遍即可。
  
 3. 最好使用独立的硬盘安装。
